## Default Permission

Default permissions for the RPC plugin

#### This default permission set includes the following:

- `allow-rpc-call`
- `allow-rpc-procedures`
- `allow-rpc-subscribe`
- `allow-rpc-unsubscribe`
- `allow-rpc-subscription-count`
